How to Use this Application:-

1.Download and Install Simple Remote Application from Website in Your Android Phone.
2.Download the Server Side Jar file fomr Website on Your PC.
3.Connect your PC(server) and Application(Client) into same lAN(Local Area Netwok.
4.Double click the Jar file to Run the Server Code for use.
5.Now In opening window,the second IP Address and Port number are Used to Connect
    your Android Application with Server.
          i.e  Your IP Address: 192.168.208.102|192.168.43.150
          PortNumber:             3000.
6.Now Open the Android Application,Enter the IP Address and Port number in it,hit 
   CONNECT button.
7.If it shows "CONNECTED" , then Use your Application and control PC wirelessly.
